### Technology Stack

- **Framework**: Phaser 3 (web game framework)
- **Language**: JavaScript (ES6+)
- **Dev Server**: Vite (instant hot reload)
- **Editor**: VS Code
- **Deployment**: GitHub Pages

**Why JavaScript?** Even though Dad knows Python, we chose JavaScript + Phaser for:
- Instant hot reload (<1 second vs 3-10 seconds)
- Built specifically for games
- Perfect for top-down adventures
- Can evolve to 3D later
